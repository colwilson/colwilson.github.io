Horizon Charts in Vue2

  • Col Wilson
  • 1 Minutes
  • May 6, 2017

I was looking for a horizon chart component for Vue2, but couldn’t find one, so I wrote my own.

Of course the point of a Horizon Chart is that it identifies change and they’re great for helping you to identify changes in metrics, for example cpu usage, dewpoint temperature, spot values etc.

So if you feed this chart live data, for example via socket-io, you’ll get nice dancing charts whizzing across the screen.

To that end I’ve created a dummy data generator and the simplest of vue2 apps to illustrate.

You can find it here.
